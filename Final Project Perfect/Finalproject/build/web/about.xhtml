Founders

Michael Leis, David Mancol, and Josh Lakota are three IT majors who took IT 353 (Web Development Technologies) at Illinois State University in the Fall of 2018 at Illinois State University.

Web App

LinkedU was a class project that was tasked by professor, Dr. Billy B. L. Lim, with the idea of connecting universities and those seeking a higher education in an efficient and simple system. LinkedU is a web application that enables users to search for universities/students who have also signed up to the application. It helps the user search for keywords they're looking for as well as helps sift through the information through a sorting feature.
